Утилита "Многочлены"
Автор: Нестерова Светлана

ОПИСАНИЕ
Данное приложение обрабатывает многочлены от многих переменных.

ОГРАНИЧЕНИЯ НА ВХОДНЫЕ ДАННЫЕ
Приложение обрабатывает выражения, состоящие из однобуквенных
переменных, действительных чисел и знаков математических операций
(+, -, *, /, ^). Выражение должно представлять из себя многочлен,
то есть быть представимым в виде суммы одночленов.
Знак умножения можно пропускать.
Допускается деление на выражение, которое упрощается до одночлена.
Допускается возвдение в степень, являющуюся выражением, которое
упрощается до целого числа (с погрешностью). При этом возведение
в отрицательную степень допустимо только для одночлена.

Примеры корректных многочленов:
xy/(x+x)
(a+b)^(x-x+100)
a^(-100)

Примеры некорректных многочленов:
(x+y)/(x+y)
(a+b)^(-100)

ТРЕБОВАНИЯ
Python версии 3

ЗАПУСК
Консольная версия - polynomials.py
Справка по запуску: python polynomials.py --help

Пример запуска:
python polynomials.py --simplify (a+b)^^10
python polynomials.py --equals (x+y)(x-y) --and x^^2 - y*y
Более коротко:
python polynomials.py -e (x+y)^100 -a (y+x)^100
При запуске из консоли не забывайте экранировать знак ^.
